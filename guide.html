User Guide

How to search customs-linked vehicle records effectively

Use this guide when you are checking a vehicle before purchase, reviewing import paperwork, or validating an amnesty-related record.

1

Start with the strongest identifier

Chassis number is usually the quickest route. If that is unavailable, move to GD, IGM, BL, ONO, or delivery order based on the documents in hand.

2

Enter the identifier exactly

Use the complete value with no accidental spaces. For GD search, fill all three fields together for better precision.

3

Use combined search when paperwork is incomplete

The combined option collects every non-empty field from the home screen and performs a broader lookup across the available source files.

4

Open the best match and review the detail sheet

From the results screen, inspect the record source and full field list, then save, print, or share the detail sheet if needed.

Do
  • Double-check the chassis number or document code before searching.
  • Compare the returned record with the seller's paperwork.
  • Keep a screenshot or printout for your own verification notes.
  • Use alternate search routes when one identifier gives no result.
Do not
  • Rely only on verbal assurances about customs clearance.
  • Ignore document mismatches between the car and the record.
  • Treat incomplete search input as conclusive evidence.
  • Skip official follow-up for legal disputes or sensitive cases.
